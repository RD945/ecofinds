# EcoFinds: A Sustainable E-Commerce Marketplace

Welcome to the official repository for **EcoFinds**, a full-stack e-commerce web application built from the ground up.  
This platform is designed as a marketplace for users to buy and sell **sustainable, eco-friendly products**, promoting a community of conscious consumerism.

## 🚀 Project Overview

EcoFinds was built to demonstrate a **complete development lifecycle**, from setting up a **custom backend and database** to integrating it with a **modern React frontend**.

---

## 🛠️ Tech Stack

- **Frontend**: React, TypeScript, TailwindCSS  
- **Backend**: Node.js, Express.js  
- **Database**: MongoDB  
- **Other Tools**: Axios, JWT Auth, Git/GitHub  

---

## 📌 Features

- User authentication and authorization  
- Product listings with category filters  
- Cart and checkout flow  
- Seller dashboard for managing products  
- Secure API with JWT  
- Responsive UI
